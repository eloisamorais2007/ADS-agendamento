Documentação Técnica: Sistema de Agendamento (Manicure) 

1. Descrição do Funcionamento 

O sistema de agendamento é voltado para manicures, permitindo que o próprio profissional defina seus horários de atendimento. Quando um cliente seleciona um horário, este fica instantaneamente indisponível para terceiros, evitando conflitos na agenda. O horário reservado só retornará ao status de "disponível" caso a manicure realize o cancelamento manual do agendamento. 

2. Política de Cancelamento e Atrasos 

Os cancelamentos automáticos ou manuais ocorrem prioritariamente em casos de atraso do cliente. Caso o atraso exceda o limite de 20 a 30 minutos, a manicure tem a prerrogativa de cancelar o atendimento para preservar a pontualidade dos compromissos subsequentes. 

3. Gestão de Equipe (Multi-profissionais) 

O sistema permite o cadastro de mais de um profissional no salão. Isso possibilita que o cliente visualize a disponibilidade global da equipe; caso um profissional esteja com a agenda lotada, o cliente pode optar por outro que esteja disponível no horário desejado. 

4. Sinal de Agendamento e Multa 

Para confirmar a reserva de um horário, a cliente deverá realizar o pagamento de um valor de sinal, definido previamente pela manicure. 

Garantia: Este valor serve como confirmação do compromisso. 

Regra de Multa: Caso a cliente desmarque o horário ou não compareça, o valor do sinal será retido como multa pelo período reservado, não havendo estorno. 

5. Acesso e Cadastro do Cliente 

O acesso à agenda e a realização de marcações são restritos a usuários cadastrados. Após fornecer seus dados, o cliente terá acesso à sua área logada para gerenciar seus agendamentos. 

Informações necessárias para o cadastro: 

Nome Completo: Para identificação e controle de presença. 

Telefone (WhatsApp): Para envio de lembretes e comunicações sobre o sinal. 

E-mail: Para login, segurança e confirmação de pagamento. 

CPF (Opcional/Recomendado): Para emissão de recibos e controle financeiro do sinal. 

Senha: Para acesso seguro ao histórico de agendamentos.
