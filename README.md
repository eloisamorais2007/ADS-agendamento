Sistema de agendamento de uma manicure  

O sistema de agendamento será para uma manicure, onde a própria profissional define os horários disponíveis. Quando uma cliente escolhe um horário, ele fica indisponível para outras pessoas, evitando conflitos na agenda. Esse horário só volta a ficar disponível se a manicure cancelar o agendamento. 

Os cancelamentos geralmente acontecem quando a cliente se atrasa. Normalmente, se o atraso for de 20 a 30 minutos, a manicure cancela o atendimento para não prejudicar os próximos horários. 

O sistema também pode incluir mais de uma manicure no salão, permitindo atender mais clientes. Assim, se a profissional desejada estiver ocupada, a cliente pode escolher outra que esteja disponível.
